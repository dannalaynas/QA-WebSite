Sign In

Open this page in Firefox only - it will not currently work in Chrome *

Note 1: This page is to demonstrate sample code only. On your own website, the Login page will be a separate page (linked from all pages only if the user is not logged in). Once the user is logged in, you would display their name above, and NOT show the login box. Instead, you would display a link to LogOut next to their name on every page.

Note 2: The spec states that you need to ask for and display both Name and Email. This sample script only uses Name - see if you can modify it to also ask for - and validate - an email address.
